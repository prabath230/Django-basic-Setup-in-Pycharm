# Django-basic-Setup-in-Pycharm
Django
Step 1 install 

1)	Open powershell 
2)	Check python version =” python --version”
3)	Check easy install python tool =”easy_install --version” (this is python app which located in “python/scripts”)
4)	Install django using easy install = “easy_install django”
5)	Check the installation = “django-admin –version”


Step 2 create django project  (you should install pycharm)

1)	go to pycharm project directory using powershell 
2)	create a project file using powershell at at that derectory = “django-admin startproject projectname”
3)	then open that project using pycharm
4)	then set interpreter 

Step 3 run django server

1)	Go to the project directory using powershell ex= C:\Users\ip\PycharmProjects\website
2)	You have to run file called manage.py = “python manage.py runserver”
 



Step 4 Creating Our First App directory
1)	Go in to the django project directory using power shell
2)	And type = “python manage.py startapp appname”
